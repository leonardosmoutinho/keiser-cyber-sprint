# NEC NP-V260 DLP Projector — Repair & Maintenance Documentation

| Spec | Detail |
|---|---|
| Model | NEC NP-V260 (Model No. NP-V260 1601691EB) |
| Type | DLP (Digital Light Processing) |

## Rear I/O Ports

- Audio In (3.5mm)
- Computer In (VGA / DB-15)
- Monitor Out COMP (VGA passthrough)
- PC Control (RS-232 DB-9)
- S-Video In
- Video In (RCA composite)
- AC Power In

## Disassembly Procedure Performed

1. **Power off and cool down** — ensured unit was fully cooled before opening
2. **Remove lamp door cover** — unscrew lamp compartment access panel (black plastic cover)
3. **Remove lamp assembly (NP13LP)** — disconnect power connector, slide out lamp module
4. **Inspect lamp** — check bulb integrity, reflector condition, connector pins
5. **Remove top housing** — unscrew all retaining screws around perimeter
6. **Remove main DLP board** — disconnect ribbon cables, antenna connectors, and power harness
7. **Remove I/O board** — disconnect all external port cables, slide out board
8. **Inspect internal components** — cooling fans, optical engine, wiring harness, lens assembly
9. **Clean interior** — remove dust from optical path and fan assemblies
10. **Reassemble in reverse order** — verify all connectors re-seated before closing housing
11. **Test** — verify lamp ignition, image output, and all I/O ports

## Key Components Inspected

| Component | Notes |
|---|---|
| Lamp (NP13LP) | Arc lamp and reflector inspected; contains mercury — handled per safety guidelines |
| DLP Main Board | TI DLP chipset, power regulation ICs, flash memory |
| I/O Board | VGA input/output, RS-232, S-Video, audio |
| Cooling Fans | 2x internal fans for lamp and optics cooling |
| Optical Engine | Lens, light path, color wheel assembly |

## Safety Notes

- NP13LP lamp contains mercury vapor — do not break; dispose per local regulations
- Allow 60+ minutes cool-down before opening lamp compartment
- Wear ESD wrist strap when handling PCBs
- Do not touch optical lens with bare hands
